# CUSTOMER-PERCEPTION-OF-AI-POWERED-PERSONALIZED-RECOMMENDATIONS-ON-SOCIAL-MEDIA-PLATFORMS
Statistical analysis of customer perception and behavior towards AI-based personalized recommendations using R
## 📌 Overview
This project analyzes how users perceive and interact with AI-based personalized recommendations on digital platforms using real survey data.

## 🎯 Problem Statement
Understanding user trust and behavior towards AI recommendations is critical for improving engagement in modern digital platforms.

## 📊 Dataset
- Sample Size: 250 respondents
- Data Type: Survey data (5-point scale)
- Variables: Accuracy, Privacy, Usefulness, Ease of Use, Perception, Behaviour

## 🔍 Key Analysis Performed
- Data Cleaning & Feature Engineering (R)
- Construct Variable Creation
- Correlation Analysis (Heatmap)
- Multiple Linear Regression
- ANOVA (Age & Education)
- t-test (Gender comparison)

## 📈 Key Insights
- Accuracy and usefulness significantly improve user perception
- Privacy concerns negatively impact perception
- Ease of use has strong positive influence on perception
- No significant demographic differences observed
- User perception strongly drives behavioral intention

## 🛠 Tools & Technologies
- R (Data Analysis & Modeling)
- Excel (Data Preparation)
- Statistical Methods (Regression, ANOVA, t-test)
